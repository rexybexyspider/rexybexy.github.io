# 🏙️ ربات شهر من - Telegram Mini App

بازی شهرسازی فارسی با گرافیک پیکسلی کلاسیک، به‌صورت Telegram Mini App.

این نسخه به‌صورت **Static** طراحی شده و روی **GitHub Pages** هاست می‌شه. نیازی به سرور بک‌اند نداره و تمام داده‌ها در `localStorage` مرورگر کاربر ذخیره می‌شن.

## ✨ ویژگی‌ها

- 🏗️ ساخت ۸ نوع ساختمان (خانه، مزرعه، چوب‌بری، معدن، بازار، سربازخانه، دیوار، قلعه)
- 🪙 تولید خودکار منابع (طلا، چوب، سنگ، غذا)
- 👥 مدیریت جمعیت و ظرفیت
- ⚔️ سیستم نبرد با غنیمت و تلفات
- 🤝 سیستم تجارت با دوستان
- 🏆 رتبه‌بندی رقابتی
- 🎨 گرافیک پیکسلی کلاسیک با انیمیشن

## 🚀 استفاده

### روش ۱: بازی مستقیم

کافیه فایل `index.html` رو در مرورگر باز کنی. تمام داده‌ها در مرورگرت ذخیره می‌شن.

### روش ۲: تنظیم به‌عنوان Telegram Mini App

برای استفاده به‌عنوان مینی‌اپ تلگرام:

1. این ریپازیتوری رو روی گیت‌هاب Pages هاست کن
2. در [@BotFather](https://t.me/BotFather):
   - بزن `/newapp`
   - رباتت رو انتخاب کن
   - عنوان: `شهر من`
   - URL: آدرس GitHub Pages (مثل `https://username.github.io/city-bot/`)
   - Short name: `mycity`
3. در رباتت بزن `/start` و دکمه‌ی بازی رو بزن!
