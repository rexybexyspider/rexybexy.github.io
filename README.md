# ربات تلگرام آب‌وهوا 🌤

رباتی ساده و کامل برای نمایش آب‌وهوا در تلگرام با استفاده از سرویس رایگان **Open-Meteo** (بدون نیاز به کلید API).

## ✨ امکانات

- 🌤 **آب‌وهوای فعلی** — دما، رطوبت، باد، بارش، وضعیت آسمان
- 📅 **پیش‌بینی ۳ روزه** — حداکثر/حداقل دما، طلوع/غروب، بارش
- ⏰ **پیش‌بینی ۲۴ ساعت آینده** — دما و احتمال بارش به‌صورت جدول
- 📍 **ارسال موقعیت** — کاربر لوکیشن می‌فرستد، آب‌وهوای همان نقطه
- ⭐ **شهرهای موردعلاقه** — ذخیره و دسترسی سریع برای هر کاربر
- 🇮🇷 **رابط کاملاً فارسی** — پیام‌ها، منوها و توصیف وضعیت آسمان
- 🌡 **واحد سلسیوس** — کیلومتر بر ساعت، میلی‌متر

## 🛠 پیش‌نیازها

- Python نسخه ۳.۱۰ یا بالاتر
- اتصال اینترنت (برای ارتباط با Open-Meteo و سرورهای تلگرام)
- توکن ربات تلگرام (از [@BotFather](https://t.me/BotFather) دریافت کنید)

## 📦 نصب

### ۱. کلون یا استخراج پروژه

```bash
unzip weather_bot.zip
cd weather_bot
```

### ۲. ساخت محیط مجازی (پیشنهادی)

```bash
python -m venv venv
source venv/bin/activate    # روی ویندوز: venv\Scripts\activate
```

### ۳. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### ۴. تنظیم توکن (اختیاری)

توکن به‌طور پیش‌فرض در `config.py` تنظیم شده است. اگر می‌خواهید آن را با متغیر محیطی بازنویسی کنید:

```bash
cp .env.example .env
# فایل .env را ویرایش کنید و توکن خود را قرار دهید
```

## 🚀 اجرا

```bash
python main.py
```

پس از مشاهده پیام `✅ ربات شروع به کار کرد`، در تلگرام به ربات خود پیام `/start` بفرستید.

## 💬 دستورات

| دستور | توضیح |
|------|-------|
| `/start` | پیام خوش‌آمدگویی و منوی اصلی |
| `/weather <شهر>` | آب‌وهوای فعلی شهر |
| `/forecast <شهر>` | پیش‌بینی ۳ روزه |
| `/hourly <شهر>` | پیش‌بینی ۲۴ ساعت آینده |
| `/add <شهر>` | افزودن شهر به علاقه‌مندی‌ها |
| `/remove <شهر>` | حذف شهر از علاقه‌مندی‌ها |
| `/favorites` | نمایش لیست علاقه‌مندی‌ها |
| `/help` | راهنمای کامل |

### مثال‌ها

```
/weather تهران
 forecast شیراز
/hourly مشهد
/add اصفهان
```

### 📍 ارسال موقعیت

در تلگرام، روی دکمه گیره (📎) کنار کادر پیام بزنید → **Location** → موقعیت خود را ارسال کنید. ربات آب‌وهوای همان نقطه را نشان می‌دهد.

## 📂 ساختار پروژه

```
weather_bot/
├── main.py              # نقطه ورودی + هندلرهای دستورات و callback
├── config.py            # تنظیمات (توکن، آدرس APIها، پیام‌های ثابت)
├── weather.py           # کلاینت Open-Meteo (geocoding + forecast)
├── storage.py           # ذخیره علاقه‌مندی‌ها در JSON
├── utils.py             # قالب‌بندی پیام‌ها و کیبوردها
├── requirements.txt     # وابستگی‌های پایتون
├── .env.example         # نمونه فایل تنظیمات محیطی
├── .gitignore
├── README.md            # همین فایل
└── data/                # دایرکتوری ذخیره علاقه‌مندی‌ها (هنگام اجرا ساخته می‌شود)
    └── favorites.json
```

## 🔧 نکات فنی

- **API استفاده‌شده**: [Open-Meteo](https://open-meteo.com) — کاملاً رایگان برای استفاده غیرتجاری
- **کتابخانه تلگرام**: `python-telegram-bot` نسخه ۲۱
- **HTTP کلاینت**: `httpx` (async)
- **ذخیره‌سازی**: فایل JSON محلی (بدون نیاز به دیتابیس)
- **تاریخ‌ها**: نمایش به فارسی با روز هفته و نام ماه میلادی

## ❓ رفع اشکال

### ربات اجرا نمی‌شود و خطای "Unauthorized" می‌دهد
توکن ربات را در `config.py` یا `.env` بررسی کنید. مطمئن شوید که توکن از [@BotFather](https://t.me/BotFather) دریافت شده باشد.

### شهر مورد نظر پیدا نمی‌شود
نام شهر را به فارسی یا انگلیسی کامل بنویسید. مثلاً `تهران` یا `Tehran`. اگر چند شهر هم‌نام وجود داشت، ربات لیستی برای انتخاب نمایش می‌دهد.

### ربات پاسخ نمی‌دهد
- مطمئن شوید اتصال اینترنت برقرار است
- لاگ‌ها را در ترمینال بررسی کنید
- اگر ربات قبلاً با webhook اجرا می‌شده، آن را با `deleteWebhook` حذف کنید
